# Content writing & strategy

Produce content that is specific, credible, and built to do a job — not generic filler. Always start from **who it's for** and **what action it should drive**.

## First, get the brief (ask only if missing)

Audience · goal/CTA · key message or offer · tone/brand voice · format & length · any required keywords or facts. If the user gave a topic but no brief, infer a sensible one and state your assumptions in one line, then write.

## Long-form & SEO articles

- **Search intent first:** decide informational / commercial / transactional, then match structure to it.
- One primary keyword + a handful of natural secondaries — woven in, never stuffed. Use real phrasing humans search.
- Structure: compelling H1, scannable H2/H3, short paragraphs, lists/tables where they help, a clear intro that states the payoff, a conclusion with a CTA.
- Add concrete detail — examples, numbers, steps. Cite sources (use `WebSearch`/`WebFetch`) for any claim or stat; don't fabricate data.
- Output a `<title>` (≤60 chars) and meta description (≤155 chars) when it's for the web.

## Landing pages & conversion copy

- Lead with the **outcome/benefit**, not features. Hero = promise + proof + single CTA.
- Sections: problem → solution → how it works → social proof → objection handling → CTA. Repeat the CTA.
- One page, one goal. Cut every sentence that doesn't move toward it.

## Email, ads & social

- **Email:** subject line earns the open; first line earns the read; one CTA. Write the subject last.
- **Ads:** hook in the first 5 words, concrete benefit, fit the platform's format/limits.
- **Social:** native voice per platform, strong opening line, no corporate stiffness.

## Voice & quality bar

- Write like a sharp human: active voice, concrete nouns, varied sentence length, no AI throat-clearing ("In today's fast-paced world…"). For tightening real-human voice on existing text, use the **humanizer** skill.
- Match the brand's tone; mirror the audience's vocabulary.
- Edit ruthlessly: every line earns its place.

## Strategy & calendars

When asked to plan: define audience + goals, pick channels, map a topic cluster (pillar + supporting pieces) around target keywords, and lay out a dated calendar (topic · format · keyword · CTA · channel). Tie each piece to a funnel stage (awareness / consideration / decision).
